# ML Project Documentation

This repository contains a sentiment analysis pipeline built with PyTorch and Transformers.

## Quick Start

1. Install dependencies: `pip install -r requirements.txt`
2. Train model: `python src/train.py`
3. Evaluate: `bash scripts/evaluate.sh`

## Project Structure

- `src/` - Source code
- `models/` - Trained models
- `datasets/` - Training data
